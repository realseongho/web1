HTML Tags

지금까지 배운 태그들은 총 17개이다.

  1. strong - 글자를 굵게!해주는 태그.

  2. u - 글자에 선을 그어주는 태그.(UnderLine)

  3. h - 겉보기엔 글자만 굵어져 보일 수 있지만 문단을 나타내는 태그.(위에 있는 HOME이 h1임)

  4. p - 단락을 나눠주는 태그.(paragraph)

  5. br - 줄바꿈 태그.

  6. img - Image사진을 첨부할 때 쓰는 태그,하지만 src(source)가 있어야 함.
    크기는 width. ex)image src="출처" width="**px"

  7. li - 위 리스트처럼 리스트를 만들 때 쓰는 자식 태그.(List)

  8. ul - 순위가 없는 aka.list의 부모 태그. (Unordered List)

  9. ol - 순위가 있는 aka.list의 또다른 부모 태그. (Ordered List)

  10. meta charset - 우리가 편집기에 한글을 적으면 웹사이트에서 인식을 못하기 때문에 예를 들어 "우리는 UTF-8이라는 언어를
    씁니다."라고 알려주는 태그. ex)meta charset="utf-8" (character set 문자규칙)

  11. title - 탭 창에 나오는 페이지에 대한 설명을 하는 태그.

  12. head - 말 그대로 머리,10,11번이 머리 즉 윗쪽에 위치하고 그걸 감싸는 태그.
    깔끔하게 보이려고 head,body,footer 3가지로 부분을 나눈다.

  13. body - head아래인 body 태그.

  14. html - 이 파일이 html파일이라는 걸 설명하는 태그.

  15. !doctype html - 이하동문.

  16. a - 링크 태그. 사용 시 ex)a href="링크주소" 처럼 쓰이고,새창으로 띄우고 싶을 땐 target="_blank"
    을 사용한다.(Anchor 닻, htmlreference 참조)

  17. input - input type="checkbox"
